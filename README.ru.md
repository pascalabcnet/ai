# ML PascalABC.NET

[English version](README.md)

**ML PascalABC.NET** — production-level платформа машинного обучения и анализа данных для PascalABC.NET.

Проект предоставляет полноценный ML-стек без внешних зависимостей: от табличного хранения данных и предобработки признаков до классических моделей машинного обучения, метрик, валидации, конвейеров и численного ядра линейной алгебры.

Цель проекта — сделать машинное обучение доступным в экосистеме PascalABC.NET, сохранив при этом строгую архитектуру, типобезопасность, предсказуемое поведение и учебную прозрачность реализации.

## Что такое ML PascalABC.NET

ML PascalABC.NET — это многоуровневая платформа машинного обучения, разработанная специально для PascalABC.NET.

Она ориентирована одновременно на несколько сценариев:

* обучение машинному обучению и анализу данных;
* университетские и школьные курсы по ИИ, ML и Data Science;
* демонстрацию алгоритмов «изнутри»;
* практическую работу с табличными данными;
* использование в составе экосистемы PascalABC.NET;
* production-ориентированное применение без внешних библиотек.

Проект не является игрушечной демонстрацией отдельных алгоритмов. Это целостная платформа с разделением слоёв, едиными интерфейсами моделей, предсказуемой семантикой операций и строгими инвариантами данных.

## Ключевые принципы

### Без внешних зависимостей

Платформа реализована на PascalABC.NET и не требует сторонних библиотек для:

* DataFrame;
* линейной алгебры;
* машинного обучения;
* предобработки данных;
* метрик;
* валидации моделей.

Это делает проект удобным для распространения, установки, изучения и включения в инсталлят PascalABC.NET.

### Production-уровневая архитектура

Архитектура проекта построена как набор слоёв с чётким разделением ответственности.

Хранение данных, предобработка, численное ядро, интерфейсы моделей, сами модели, метрики, валидация и конвейеры не смешиваются друг с другом.

### Учебная прозрачность

Алгоритмы и структуры данных реализованы так, чтобы их можно было изучать, объяснять и использовать в преподавании.

Код не должен быть «магией». Важная часть проекта — показать, как устроены DataFrame, ML-модели, конвейеры, метрики и численные алгоритмы изнутри.

### Предсказуемое поведение

Платформа избегает скрытых неявных преобразований.

Некорректные входные данные, рассогласование схемы, ошибки типов, неподдерживаемые ключи, неинициализированные состояния моделей и другие нарушения контрактов должны обнаруживаться явно и как можно раньше.

### Типобезопасность

Типы столбцов, схемы данных, метки классов, матрицы, векторы, модели, препроцессоры и конвейеры имеют явные контракты.

Это особенно важно для PascalABC.NET как статически типизированной образовательной и практической среды программирования.

## Архитектура

Платформа имеет слоистую архитектуру:

```text
DataFrameABC
    ↓
DataAdapters
    ↓
MLABC
    ↓
MLModelsABC / ValidationML / MetricsABC
    ↓
LinearAlgebraML
```

## Основные модули

### DataFrameABC

`DataFrameABC` — слой табличного хранения и обработки данных.

Он реализует column-store DataFrame с типизированными столбцами:

```text
Int / Float / Str / Bool
```

Основные возможности:

* типизированное хранение столбцов;
* явная схема данных;
* поддержка пропущенных значений через validity masks;
* Cursor API для построчного доступа;
* фильтрация строк;
* выбор и удаление столбцов;
* переименование столбцов;
* группировка;
* соединения таблиц;
* сохранение metadata;
* поддержка categorical flags;
* предсказуемая семантика табличных операций.

`DataFrameABC` является базовым слоем платформы: именно он отвечает за представление табличных данных и подготовку их к дальнейшей обработке.

### DataAdapters

`DataAdapters` связывает табличный слой с численным ML-слоем.

Этот модуль преобразует столбцы `DataFrame` в матрицы и векторы, которые используются моделями машинного обучения.

### LinearAlgebraML

`LinearAlgebraML` — численное ядро платформы.

Оно предоставляет плотные матрицы и векторы, а также операции и алгоритмы, необходимые для машинного обучения.

В этот слой входят:

* `Matrix`;
* `Vector`;
* базовые операции линейной алгебры;
* численные разложения;
* PCA;
* низкоуровневая численная основа для моделей.

### MLCoreABC

`MLCoreABC` определяет общие интерфейсы моделей и преобразований:

```text
IModel
ISupervisedModel
IClassifier
IRegressor
IProbabilisticClassifier
ITransformer
IPreprocessor
```

Эти интерфейсы задают единый контракт для моделей, препроцессоров, трансформеров, конвейеров, метрик и валидации.

### MLModelsABC

`MLModelsABC` содержит классические модели машинного обучения.

В платформу входят:

* линейная регрессия;
* логистическая регрессия;
* k-nearest neighbors;
* деревья решений;
* случайные леса;
* градиентный бустинг;
* классификационные модели;
* регрессионные модели.

Модели реализованы без обращения к внешним ML-библиотекам и работают через общие интерфейсы платформы.

### PreprocessorABC

`PreprocessorABC` содержит средства предобработки данных и feature engineering.

Поддерживаются:

* заполнение пропущенных значений;
* стандартизация;
* min-max scaling;
* label encoding;
* one-hot encoding.

Препроцессоры используют логику `Fit` / `Transform` и могут применяться как отдельно, так и в составе ML-конвейеров.

### ValidationML

`ValidationML` отвечает за проверку качества моделей.

Он включает:

* K-fold cross-validation;
* stratified K-fold cross-validation;
* вспомогательные средства для кросс-валидации.

Валидация работает через общие интерфейсы моделей и механизм клонирования конфигурации модели.

### MetricsABC

`MetricsABC` предоставляет метрики для задач классификации и регрессии.

Слой метрик ориентирован на явную работу с предсказаниями, encoded labels, confusion matrix и стандартными показателями качества моделей.

### Pipelines

Платформа содержит систему конвейеров для построения полных ML-сценариев.

Конвейеры поддерживают:

* предобработку данных;
* обучение моделей;
* предсказание;
* вероятностное предсказание для классификаторов;
* кодирование целевых меток;
* DataFrame-based workflow;
* matrix-based workflow.

Специализированные конвейеры используются для задач классификации, регрессии и кластеризации.
