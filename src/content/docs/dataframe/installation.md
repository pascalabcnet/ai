ML PascalABC.NET входит в состав PascalABC.NET начиная с версии 3.11.1.3833.

Если PascalABC.NET ещё не установлен, скачайте последнюю версию с [официального сайта](https://pascalabc.net/) и установите её.

После установки доступны:

- DataFrame для работы с табличными данными;
- средства подготовки данных;
- алгоритмы машинного обучения;
- метрики качества;
- инструменты визуализации.

Дополнительные внешние библиотеки устанавливать не нужно.

## Проверка установки

Создайте новый файл и выполните программу:

```pascal
uses MLABC;

begin
  var df := DataFrame.FromCsvText('''
  Имя,Возраст
  Анна,20
  Борис,25
  ''');

  df.Print;
end.
```

Если программа успешно скомпилировалась и запустилась, библиотека готова к работе.
